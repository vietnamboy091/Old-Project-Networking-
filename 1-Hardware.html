Hardwaregeräte

Generell Client:

• Handelt sich um einen Computer/ Software
• Mit dem Server kommuniziert
• Bezeichnet als Clientanwendung/-programm
• Anfragen an den Server zu übermitteln
• Erfüllt selbst keine Serveraufgaben, dient zur Schnittstelle
• Beispiele: E-Mail-Programme, Browser, FTP-Clients

Generell Server:

• Hardware --> Computer bezeichnet --> Ressourcen zur Verfügung stellt
• Host --> für hardwarebasierten Server verwendet
• Meint ein Computerprogramm
• Stellt meist Funktionalitäten wie Dienstprogramme und Netzwerke zur Verfügung
• Ermöglicht den Zugang zu Daten

Definition:

• Client und Server, keine Hardware --> Rollen beschreiben
• Aufgabe --> bestimmte Dienste bereitzustellen
• Nimmt Anfragen von Clients entgegen, verarbeitet sie
